ALGUMA COISA
O GitHub é uma plataforma baseada em nuvem em que é possível armazenar, compartilhar e trabalhar com outras pessoas para escrever códigos.

O seguinte é possível ao armazenar códigos em um "repositório" no GitHub:

Demonstre ou compartilhe seu trabalho.
Acompanhe e gerencie alterações no código ao longo do tempo.
Permita que outras pessoas revisem o seu código e deem sugestões para melhorá-lo.
Colabore em um projeto compartilhado sem se preocupar que suas alterações afetarão o trabalho de seus colaboradores antes que você esteja pronto para integrá-los.

O GitLab é uma plataforma completa de DevSecOps baseada no sistema de controle de versão Git. Diferente de ferramentas que focam apenas na hospedagem de código, o GitLab integra em uma única aplicação todo o ciclo de vida do desenvolvimento de software — desde o planejamento inicial até o monitoramento em produção.

O Bitbucket serve como um serviço de hospedagem de repositórios baseado na nuvem que permite a equipes de desenvolvimento gerenciar e colaborar em seus códigos-fonte. Ele é mantido pela Atlassian e foca principalmente no controle de versão e na integração com outras ferramentas de gestão de projetos. 

Um Pull Request (PR) é uma solicitação formal para mesclar um conjunto de alterações de código de uma ramificação (branch) para outra, geralmente a principal, permitindo que outros desenvolvedores revisem, discutam e aprovem as mudanças antes que sejam integradas ao projeto, essencial para o desenvolvimento colaborativo e a manutenção da qualidade do código. Ele funciona como uma proposta para "puxar" suas alterações para o código base principal, sendo fundamental para fluxos de trabalho em equipe. 

Issue tracking (rastreamento de problemas) é um processo e sistema de software utilizado para registrar, priorizar, monitorar e resolver bugs, tarefas ou solicitações de suporte. Essencial para o desenvolvimento de software e atendimento ao cliente, ele centraliza o histórico de atividades, permitindo que equipes colaborem na resolução de problemas de forma organizada.
